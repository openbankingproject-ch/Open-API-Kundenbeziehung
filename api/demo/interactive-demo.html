OBP Interactive Demo - 13-Step OAuth Process

Welcome to the Interactive Demo

This demo showcases the complete 13-step customer onboarding process with OAuth 2.0 integration:

  • Steps 1-2: Pre-authorization (Initialization & Product Selection)
  • Steps 2a-2c: OAuth Flow (Authentication → Consent → Token)
  • Steps 3-10: Resource Access (All data collection with Bearer token)

You'll see how a customer opens an account at Bank B while securely sharing their existing data from Bank A using OAuth 2.0.
